\\\ Инструкция к применению \\\
1. Создать папку Music
2. В папку Music положить треки, которым нужно выставить тег
2. Положить файл script.exe рядом с папкой Music
3. Запустить файл script.exe
4. Выбрать опцию путем написания соответствующей цифры в меню и нажать Enter
5. Дождаться завершения работы скрипта
6. Для завершения работы скрипта используйте пункт 0.
